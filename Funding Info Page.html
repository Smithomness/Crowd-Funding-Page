    Spending Plan

    The spending will mostly be directed torwards building the backend or functionality for my website. Such as API's, javascript for cart and shopping funcitons, setting up a database to maintain and store customer information securely. Setting up payment processing and tracking for customers. Essentially setting up the functionality of the website will likely be where %70 - %80 of funding will go to. The rest will be put towards our first marketing campaign. Getting pictures, videos, hiring models to wear our products, posting and running ads on social media sites and basically anything that comes along with promoting and marketing our brand. So far I've been a one man show. I've been learing everything this far when it comes to building this website and I will continue to do so. The problem is time. How much time would it take for me to learn sending https request, setting up API's, serverside code languages like nodejs or php, using javascript for shopping cart functions, setting up Stripe so customers can make secure payments. There is just so much that come's along with building a website. Most people see and interact with the front-end of the website but the back-end is what makes those interactions seemless.
